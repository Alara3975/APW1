January 2021 - Color Palette Generator

My most recent project is a random color palette generator. I presented on it as well as the history of colors in the programming world. My presentation should be above this text. For the generator, I used JavaScript as well as HTML to make it. It creates a randomly generated, non-coordinated palette of 5 different colors. I unfortunately could not get the hex codes to show without breaking the generator. To expand off what I have done so far, I could find a way to get the colors of the palettes to actually aesthetically fit together.

June 2020 - Translator

My project was a English to Spanish/Spanish to English translator in Python. I worked on this project with Leila last year. We used dictionary APIs from Merriam Webster to create a translator that will take a word in one language and translate it to the other language. As an expanding point from that, we would have started to try and make our translator translate whole sentences or phrases.
