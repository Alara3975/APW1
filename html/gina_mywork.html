Journal 2: Possible Project?
February 21st, 2021

I talked to Dr. Henry about my ideas for a coding project and she happened to be giving a presentation to teachers on implementing programming into other classes. In her physics class, we often use code to model real life experiments, which has been especially useful during Covid. I think going virtual has pushes other classes to use more virtual and online resources.
Though I was considering finishing a game I started a very long time ago, lately I've been leaning towards more useful and practical projects.
