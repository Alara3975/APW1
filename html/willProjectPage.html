For my project, I'd like to make a multiplayer game. I would first learn some basics of C# and how to use the software "Unity". I'd like to recreate / make something similar to a game called League of Legends. It is third person with the camera looking down upon your character. Here is some example gameplay footage: League of Legends gameplay. It's a 5 versus 5 fight where the goal is to destroy the enemy's base. There are turrets that block your path though. They will shoot you if you get too close. Small computer controlled characters called "minions" will walk towards the turrets and take the damage for you while you destroy it. The enemy will also have minions so you have to kill the enemy's and push before they do the same. You can choose a character to play as, each one having unique abilities. You gain gold by killing minions and other players, which you can use to buy items that will make you stronger.

I think making a smaller version of this could be a fun challenge for me. I like 3D modeling as well as coding so this project could combine two things I enjoy which would motivate me to work on it more often. Some goals I have for this project would be to make at least 6 unique characters so that there is a small selection to choose from which might make it feel more complete. I'd also like to be able to send this to, and play it with some friends. My main goal by the end of this project is to be able to confidently code in C#.
